# Public Assets

This folder is for your static assets. 
To update the images on the website, please upload your images here with the following filenames:

- `hero1.jpg` (First image in the hero slideshow)
- `hero2.jpg` (Second image in the hero slideshow)
- `hero3.jpg` (Third image in the hero slideshow)
- `before.jpg` (The "Before" image in the transformations slider)
- `after.jpg` (The "After" image in the transformations slider)

The website is already configured to look for these exact filenames in this folder. If the files are not found, it will temporarily fall back to placeholder images.
